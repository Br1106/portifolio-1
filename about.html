Sobre mim

Hoje busco uma oportunidade de ingressar na carreira de desenvolvedor. Tenho noções de Orientação a objeto, API Rest e banco de dados . Estou me aprofundando atualmente em programação Java com Frameworks como Spring Boot e JPA. No momento estou cursando inglês intermediário. Também quero destacar que na faculdade tive contato com linguagem C para sistemas embarcados
